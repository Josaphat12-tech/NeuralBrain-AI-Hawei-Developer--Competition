API Documentation

Comprehensive REST API reference for NeuralBrain-AI health analytics platform

Base URL

All API endpoints are relative to:

https://neuralbrain-ai.herokuapp.com/api

For local development: http://localhost:5000/api

Core Endpoints

GET
/api/status
Get system status and available endpoints
Returns:
✓ API status (operational/maintenance)
✓ System uptime and version
✓ Available endpoints summary
{
  "status": "operational",
  "app_name": "NeuralBrain-AI",
  "version": "1.0.0",
  "timestamp": "2026-01-29T12:00:00Z"
}
GET
/api/health
Simple health check endpoint for monitoring
{
  "status": "healthy",
  "timestamp": "2026-01-29T12:00:00Z"
}
POST
/api/ingest
Trigger data ingestion from all configured health APIs
{
  "status": "success",
  "message": "Ingestion complete: 2 source(s) succeeded",
  "total_apis": 2,
  "successful": 2,
  "failed": 0,
  "execution_time_seconds": 2.34
}
GET
/api/data
Retrieve normalized health data with optional filtering and pagination
Query Parameters:
source (string): Filter by data source
status (string): Filter by status (valid/invalid/pending)
limit (int): Max records to return (default: 100)
offset (int): Starting position (default: 0)
{
  "status": "success",
  "total": 155,
  "returned": 50,
  "data": [...]
}
GET
/api/data/<id>
Retrieve a specific health data record by ID
GET
/api/logs
Get ingestion operation logs and audit trail
Query Parameters:
source (string): Filter by API source
status (string): Filter by log status
limit (int): Max logs (default: 50)
POST
/api/validate
Validate health data against defined rules and constraints
{
  "valid": true,
  "errors": [],
  "warnings": []
}

HTTP Status Codes

200 OK
Request successful
201 Created
Resource created successfully
400 Bad Request
Invalid parameters or request body
401 Unauthorized
Missing authentication or invalid token
404 Not Found
Resource does not exist
500 Error
Internal server error

Getting Started

Here's a quick example to get started with the API:

# Python Example
import requests

response = requests.get('http://localhost:5000/api/health')
data = response.json()
print(data)
